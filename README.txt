Чтобы воспользоваться(протестировать работу) приложения, необходимо сделать следующее:
1)Скачать подходящую Вам версию MariaDB (https://downloads.mariadb.org/mariadb/10.5.10/).
2)Установить MariaDB, указав следующие параметры при установке:
  2.1)Пароль: 269973514.
  2.2)Порт: 3307.
3)Запустить MariaDB, создать базу данных с именем "users", для этого нужно после открытия сеанса нажать на имя сеанса правой кнопкой мыши, 
затем выбрать Создать/База данных. Имя задать как "users", сопаставление кодировок оставить по умолчанию.
4)Скачать архив, распаковать его. В папке main запустить файл main.exe. Готово! 

Можно поместить все файлы репозитория, кроме папки main и readme.txt, в одну директорию и запустить через любую ide. Однако в этом случае, 
предварительно так же нужно произвести действия 1-3 + необходимо будет скачать и установить использующиеся в приложении библотеки (PyQt5, PyMySQL). 

Краткое руководство пользователя приложения: CRUD-приложение с акцентом на две таблицы бд (MariaDB): таблица с данными учетных записей (логин + пароль) 
и таблица с контактными данными (полное имя + номер телефона + дата рождения).
Имеются функции напоминания(ближайшие даты рождения на неделю), автовхода при включенном чекбоксе и непосредственно функции CRUD: создание контактов 
происходит при нажатии кнопки и заполнении полей. Редактирование и удаление происходит посредством нажатия на ячейку таблицы левой кнопкой мыши. 
Сами данные отображаются в виде таблицы с разделением на страницы по буквам русского алфавита.
